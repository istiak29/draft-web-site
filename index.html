How to push in github repo for the first time:

  1. open cmd
  2. go to the main directory
  3. type git init
  4. git add . (for adding all file)
  5. git commit -m"add a comment" (for tracking your changes)
  6. git branch -M main
  7. git push -u origin main

How to add new changes in github repo:

  1. git add .
  2. git commit -m"add a meaningful comment"
  3. git push
